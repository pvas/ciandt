# ciandt
Case Tecnico


Para inicializar o webservice, deverá ser utilizado:

IDE Eclipse;
Servidor de aplicação Tomcat 8;
Banco de dados H2;
Maven.

Importar o projeto no eclipse e fazer o deploy no Tomcat, que deverá ser inicializado.

A URL de conexão do webservice é: 

 - localhost:8080/CIandT/api/carregarRotas
 - localhost:8080/CIandT/api/buscarMelhorRota
 
 Para carregar o arquivo malha.txt no banco de dados, deve-se executar a classe LogisticaClient.java.
